Tinkerkit Arduio based audio player

### Install 

1\. Install SoX. For OSX it's best to use Homebrew

```
brew install sox
```

2\. Install modules

```
npm install duino socket.io tako
```

3\. Create ```files``` subdirectory and copy some audio files there

4\. Run

```
node app.js
```

5\. Point browser to

```
client/index.html
```
